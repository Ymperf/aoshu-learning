# 小学奥数学习应用 - 部署命令清单

## 前提条件
- 已安装 Git
- 已有 GitHub 账号

## 第一步：在 GitHub 创建仓库

1. 访问 https://github.com/new
2. 填写信息：
   - Repository name: `aoshu-learning`
   - Public
   - 不要勾选任何初始化选项
3. 点击 **Create repository**
4. 复制仓库地址（例如：`https://github.com/你的用户名/aoshu-learning.git`）

## 第二步：执行以下命令

在项目目录 `e:\CC-Project\aoshu` 下打开命令行（Git Bash 或 PowerShell），依次执行：

### 1. 初始化 Git（如果还没有）
```bash
git init
```

### 2. 配置 Git 用户信息（首次使用需要）
```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱@example.com"
```

### 3. 添加所有文件
```bash
git add .
```

### 4. 创建提交
```bash
git commit -m "完整版：小学奥数学习应用（含800道题+SVG图形）"
```

### 5. 关联远程仓库
**替换下面的地址为你的仓库地址：**
```bash
git remote add origin https://github.com/你的用户名/aoshu-learning.git
```

### 6. 推送到 GitHub
```bash
git branch -M main
git push -u origin main
```

如果弹出浏览器要求授权，点击授权即可。

## 第三步：在 Vercel 部署

### 方式一：网页操作（推荐）

1. 访问 https://vercel.com/
2. 点击 **Continue with GitHub** 登录
3. 点击 **Add New...** → **Project**
4. 找到 `aoshu-learning` 仓库，点击 **Import**
5. 配置项目（通常自动检测，无需修改）：
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. （可选）添加环境变量：
   - 点击 **Environment Variables**
   - Name: `ANTHROPIC_API_KEY`
   - Value: 你的 Claude API Key
7. 点击 **Deploy**

等待 2-3 分钟，部署完成！

### 方式二：命令行操作

```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login

# 部署
vercel

# 生产环境部署
vercel --prod
```

## 后续更新流程

每次修改代码后，只需三个命令：

```bash
git add .
git commit -m "描述你的修改"
git push
```

Vercel 会自动检测并重新部署。

## 常见问题解决

### 问题1：git push 提示需要认证

**解决方案1：使用 GitHub CLI（推荐）**
```bash
# 安装 GitHub CLI: https://cli.github.com/
gh auth login
```

**解决方案2：使用 Personal Access Token**
1. 访问 https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 勾选 `repo` 权限
4. 生成后复制 token
5. 推送时使用 token 作为密码

### 问题2：git commit 失败，提示需要配置用户信息

```bash
git config --global user.name "你的名字"
git config --global user.email "你的邮箱"
```

### 问题3：git remote add 提示 remote origin already exists

```bash
# 删除旧的远程仓库
git remote remove origin

# 重新添加
git remote add origin https://github.com/你的用户名/aoshu-learning.git
```

### 问题4：Vercel 部署失败

检查：
- `package.json` 中是否有 `"build": "tsc && vite build"`
- Node.js 版本是否兼容（推荐 18.x 或 20.x）
- 查看 Vercel 部署日志中的具体错误

## 验证部署成功

访问 Vercel 提供的域名（如 `https://aoshu-learning.vercel.app`），检查：
- ✅ 首页正常显示
- ✅ 可以浏览专题
- ✅ 可以做练习题
- ✅ SVG 图形正常显示

## 完整命令速查表

```bash
# 首次部署
git init
git add .
git commit -m "初始提交"
git remote add origin https://github.com/用户名/仓库名.git
git branch -M main
git push -u origin main

# 后续更新
git add .
git commit -m "更新说明"
git push

# 查看状态
git status
git log --oneline -5

# 撤销修改（慎用）
git checkout -- 文件名        # 撤销单个文件
git reset --hard HEAD         # 撤销所有修改

# 查看远程仓库
git remote -v

# 更改远程仓库地址
git remote set-url origin 新地址
```
