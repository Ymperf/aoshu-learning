# 小学奥数学习应用

完整的小学奥数学习 Web 应用，包含：
- 6个年级，80个专题
- 800道练习题（选择题、填空题、判断题、步骤题）
- 20+ SVG 图形辅助理解
- 进度追踪（localStorage）

## 技术栈
- React 18 + TypeScript + Vite
- Tailwind CSS
- Zustand (状态管理)
- React Router v6
- react-markdown + rehype-raw (SVG 渲染)

## 本地运行
```bash
npm install
npm run dev
```
