## 图形设计规范

### 颜色方案
- **主线条**：`#2563eb`（蓝色）
- **标注文字**：`#dc2626`（红色）
- **填充色**：`#dbeafe`（浅蓝）、`#fef3c7`（浅黄）
- **辅助线**：`#94a3b8`（灰色）
- **特殊标记**：`#22c55e`（绿色）、`#fbbf24`（黄色）

### 尺寸规范
- **宽度**：300-400px
- **高度**：根据内容调整，通常100-280px
- **stroke-width**：2-3px
- **font-size**：12-14px（标注）

### 技术特点
- 使用 `viewBox` 确保响应式
- 所有文字使用中文
- 添加 `text-anchor="middle"` 居中对齐
- 使用 `<defs>` 定义可复用元素（如箭头）

## 教学价值

### 帮助理解抽象概念
- **空间关系**：通过图形直观展示位置、方向、距离
- **数量关系**：用颜色和标注区分不同对象
- **变化过程**：展示图形变换、运动轨迹

### 降低认知负担
- **视觉化**：将文字描述转化为图形
- **结构化**：清晰的标注和分类
- **记忆点**：颜色和形状帮助记忆

### 适应不同学习风格
- **视觉学习者**：通过图形理解
- **逻辑学习者**：结合图形和推理
- **动手学习者**：可以模仿绘制
