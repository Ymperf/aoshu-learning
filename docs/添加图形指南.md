# 为专题添加图形说明

本应用已集成 react-markdown，支持在讲解内容和例题中嵌入 SVG 图形。

## 使用方法

在 `src/data/contents/gradeN.ts` 文件中，可以在 `explanation` 或 `examples` 的 `problem`/`solution` 字段中直接嵌入 SVG 代码。

### 示例1：三角形计数

```typescript
problem: `数一数下图中共有多少个三角形？

<svg width="200" height="173" viewBox="0 0 200 173" xmlns="http://www.w3.org/2000/svg">
  <!-- 大三角形外框 -->
  <polygon points="100,20 20,160 180,160" fill="none" stroke="#2563eb" stroke-width="3"/>
  <!-- 内部分割线 -->
  <line x1="100" y1="20" x2="100" y2="160" stroke="#2563eb" stroke-width="3"/>
  <line x1="20" y1="160" x2="100" y2="90" stroke="#2563eb" stroke-width="3"/>
  <line x1="180" y1="160" x2="100" y2="90" stroke="#2563eb" stroke-width="3"/>

  <!-- 标注 -->
  <text x="60" y="130" font-size="14" fill="#dc2626">①</text>
  <text x="130" y="130" font-size="14" fill="#dc2626">②</text>
  <text x="95" y="60" font-size="14" fill="#dc2626">③</text>
</svg>`
```

## 适合添加图形的专题类型

### 几何类专题
- **图形认识**：基本图形展示
- **图形拼组**：拼接过程演示
- **周长与面积**：图形分割示意
- **图形计数**：复杂图形结构
- **立体图形**：展开图、三视图

### 计数类专题
- **植树问题**：树木排列示意
- **火柴棒问题**：火柴摆放图
- **路径计数**：网格路径图

### 逻辑类专题
- **天平问题**：天平状态图
- **方向与位置**：方位示意图
- **推理问题**：关系图、表格

### 应用题类专题
- **行程问题**：路线示意图
- **工程问题**：进度条、时间轴
- **浓度问题**：容器示意图

## SVG 常用元素

### 基本形状
```xml
<!-- 矩形 -->
<rect x="10" y="10" width="100" height="50" fill="none" stroke="#2563eb" stroke-width="2"/>

<!-- 圆形 -->
<circle cx="50" cy="50" r="30" fill="none" stroke="#2563eb" stroke-width="2"/>

<!-- 线段 -->
<line x1="0" y1="0" x2="100" y2="100" stroke="#2563eb" stroke-width="2"/>

<!-- 多边形 -->
<polygon points="50,10 90,90 10,90" fill="none" stroke="#2563eb" stroke-width="2"/>

<!-- 文字 -->
<text x="50" y="50" font-size="14" fill="#dc2626">标注</text>
```

### 颜色建议
- 主线条：`#2563eb`（蓝色）
- 辅助线：`#94a3b8`（灰色）
- 标注文字：`#dc2626`（红色）
- 填充：`#dbeafe`（浅蓝色）或 `none`（透明）

## ASCII 艺术图（简单场景）

对于非常简单的图形，也可以使用 ASCII 艺术：

```typescript
problem: `观察下面的图形规律：

    ○ △ □ ○ △ □ ○ △ ___

下一个图形是什么？`
```

## 注意事项

1. **SVG 尺寸**：建议宽度不超过 400px，以适配移动端
2. **viewBox**：使用 viewBox 确保图形可缩放
3. **stroke-width**：线条粗细建议 2-3px
4. **文字大小**：font-size 建议 12-16px
5. **颜色对比**：确保在白色背景下清晰可见
6. **无障碍**：为复杂图形添加文字描述

## 批量添加建议

优先为以下专题添加图形：
1. 一年级：数数与计数、图形认识、火柴棒问题
2. 二年级：植树问题、图形拼组、方向与位置
3. 三年级：鸡兔同笼、周长与面积、图形计数
4. 四年级：面积计算、三角形与四边形
5. 五年级：立体图形、圆的初步
6. 六年级：圆与扇形、几何综合
